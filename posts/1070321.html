顏色辨識大突破

製作解魔術方塊機器

Posted by Dowios on March 21, 2017

之前在製作顏色辨識時，最嚴重的問題，莫過於顏色在不同光線底下無法正確辨識。

但如今終於有方法可以解決了。

為魔術方塊加上燈光

為魔術方塊加上燈光，這樣即可讓光線維持在各面平均的情況。

之前沒有這樣做的原因是，光線若太亮攝影機會曝光過度。但現在我們找到了方法來調整攝影機曝光值，因此這個方法也變為可行。

改進方法：

1.為魔術方塊加上燈光。3D列印一個燈光架，讓燈光和攝影機朝向同樣的三個面。

2.安裝Webcam的驅動程式，並且兩隻個別進入設定頁面將曝光調成手動，且調至最低。

如此，顏色辨識的問題就被改善了。

為顏色辨識建立標準

現在，因為攝影機的參數已經固定，燈光的色溫也都已經被打光的燈固定了，所以我們可以訂定一套實際的顏色範圍，再也不需要隨著環境變化。

以下為測出的顏色範圍

  if px[1] < 110 and dis > 20:
    return "W" //先看亮度和彩度判別白色(dis = V - S)
  elif 58 > px[0] >= 20:
    return "Y"
  elif 90 > px[0] >= 58:
    return "G"
  elif 160 > px[0] >= 90:
    return "B" //藉由色相辨別容易辨別的Y,G,B
  else:
    if 5 < px[0] <= 160:
      return "O"
    elif px[0] > 160 or px[0] < 3 :
      return "R"
    else: //針對難判別的部分(H = 3~5)用亮度作進一步判別
      if 100 > px[2]:
        return "R"
      else:
        return "O"
